# Habit Tracker

Aplikasi web untuk melacak kebiasaan harian, dibangun dengan React.

🔗 **Demo langsung:** https://habit-tracker-beige-two.vercel.app/

<img width="1920" height="1080" alt="Cuplikan layar dari 2026-07-01 21-27-00" src="https://github.com/user-attachments/assets/aab56918-4514-4312-91a0-1052c284a25d" />

## Fitur

- Menandai kebiasaan per hari (Senin–Minggu)
- Progress harian (mis. 0/2 selesai)
- Tambah / hapus kebiasaan
- Catatan per kebiasaan
- Mode gelap
- Ganti background

## Teknologi

- React
- JavaScript
- CSS
- Vite

## Menjalankan secara lokal

```bash
# clone repositori
git clone https://github.com/anggaJavaBackEndDev/habit-tracker.git
cd habit-tracker

# install dependency
npm install

# jalankan mode development
npm run dev
```

Buka `http://localhost:5173` di browser.
